# Bank-Account-System
A Bank Account Management System in Python is a terminal-based application that automates core retail banking operations like account creation, fund deposits, cash withdrawals, and balance tracking. 

A Python OOP project with:

- Account Creation
- Deposit
- Withdraw
- Transfer
- Transaction History
- JSON Storage
- Auto Save
- Auto Load
- Timestamp Tracking

## Technologies

- Python
- JSON
- OOP
- File Handling

## Features

- Unique Account Validation
- Minimum Balance Rule
- Transaction IDs
- Last N Transactions
